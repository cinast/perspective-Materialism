# 此仓库

是一个文献库，  
存放着我几个月正在思考的一种元世界观理论（暂且让我这么定位吧）：

$$
\fbox{
                视界唯物主义论
\\        perspective - Materialism
}
$$

我原先是思考长历史跨度应该如何预测，  
因为人类历史不可能到共产主义之后就没有后续了，  
而且物质这条线虽然说不会立即退出历史主旋名单，但是会降级到次要矛盾，甚至是次次次要到不需要关心了。

人类以后会走什么路线呢？这里就只能留个空。
