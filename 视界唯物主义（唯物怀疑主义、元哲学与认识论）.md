## 第一章：世界观和元框架——我们需要一种抽象的构造方式

### 1.1 元始：元框架原本是空的

有一个东西，我们永远写不出来。  
它没有内容，没有立场，没有概念，连“空”这个字都**不应该有**。

我们叫它**元始**。
你可以把它想象成数学里的“空集”——但你不能指着它说“这就是空集”，但你知道它在那里，否则你无法定义零。  
换句具体的话来说，就是个平凡的占位符。  
它是所有可能世界的逻辑起点，但它本身不是任何世界，不可言说。

那为什么我们需要这个“占位符”？  
因为我们需要他容纳一切，我们需要用它理解这个理论在遇到不同的使用者和不同的条件下，它会如何变化。

由此我们需要描述，我们不可能对某人说出“空气”，  
论实际上，我们总是预先加入了某种预设，这些预设来自我们的属性，  
他们奠基了我们如何观察世界，如何认识世界，如何描述世界，如何假设世界。

于是“元始”在我们这里，将不再是元始，而先是我们自己，  
我们自己零散的认识，将组织为一套世界观，
接着我们在世界观上挖点空，然后也许再加点抽象的条件，  
而且也只能这么干，一切假设源自视界（见“视界”），  
于是**元框架**从此诞生。

为什么需要他？我们需要足够的空间，允许假设，扩张视界。  
这一点我在稍后的章节中会逐步解释。

然后，元始我只提一次，然后闭嘴。  
后面所有的章节，都不再提元始，除非需要那个占位符，而是说元框架。

这本书，本身就是元框架的一种，但是因为是指南性质，抽象层级略高一头，  
不过你还可以用它制造一点略微具体化的元框架世界观机器或者什么具体世界观之类的。

### 1.2 抽象层级

元框架不好直接指代，但我们可以描述它的结构。它包含若干个抽象层级：

- 0.元始：纯粹逻辑占位符
- **I. 使用者具体化的元构造层**：当某个使用者（比如你）把自己的属性嵌入元构造层后，生成的带有具体性质的框架。
- **I\*. 方法论层**：本论自身的指导逻辑——也就是你现在正在读的这份文本。
- **II. 具体实例层**：用本论构造出的略微具体化的元构造机器或者具体的世界观、主义、描述集。

**关键区分**：  
0 层不可写，不可说，只是逻辑的占位符。  
I 层是每个使用者自己心里的框架。  
I\* 层是这本书的内容。  
II 层是你用这本书去构造的东西。

混淆不同层级，是思维混乱的主要来源。  
警示第一条：**不要混淆层级**。

## 第二章：视界——那条你永远跨不过去的边界

我们的读者，也就是使用者，几乎在我的视界内，没有一个不具有“观”这个属性的，“视”的属性更不会缺失。

### 2.1 视界的定义

**视界**：对于某个观测者而言，可观测、可描述、可接触的范围的抽象意义集合。

- 视界之内：可观测、可描述。
- 视界之外：对于该观测者是“尚未添加约束”的状态——不可知、不接触、不可描述。  
   若果需要逻辑上的占位符，那其内容，就是“元始”一伦的东西，毫无约束，不能假定。

一定是视界是世界的子集（世界 ≥ 视界）。  
世界是本体论意义上的全体，视界是认识论意义上的子集。

而且无论如何，你总是处于视界中，因为你总是在认识周遭，并且一直着感觉它。  
如果强行把你和视界隔离了，你也就不会与他们有认识意义的接触了，至少对你来说。

然而，如果用具体的概念与视界一词组合，就可以说：

> 那地方我不认识，那地方在我视界之外。

（我在这个地方这个时间所处的视界，看到了那个地方出现在我的视界里，但是我只看到了它的一部分，它存在，但是没有看到它的全部部分，没有看到它出现的原因，我不认识它）

### 2.2 观测者模型

> 由第零章得知，我们的“观”属性摆脱不掉，而且是根本的属性。

我们构建模型三要素：**世界、望远镜、观测者**。

- **世界**：被观测的对象整体。必须被视为绝对全集——涵盖一切内容。如果有超出当前认知范围的事物，那不是“世界的遗漏”，而是观测者的视界有限。
- **望远镜**：观测工具。包括物质工具（显微镜、探测器）和思想工具（语言、理论、意识形态、数学框架）。望远镜决定**如何看**。
- **观测者**：进行观测的主体，**在世界之内**。如果观测者在世界之外，那他就与世界没有关系，也就不存在“观测”一说。

观测者自身的性质影响视界：

- 物理性质：位置、运动状态、感官能力
- 认知性质：知识储备、思维模式、先验假设
- 历史性质：所处的时代、文化背景、个人经历

**动态性**：三者相互影响

1. 观测者改变望远镜 → 视界扩展
2. 望远镜改变观测者 → 视界重构
3. 世界反作用于观测者 → 视界被迫移动

**递归性**：观测者可以观测另一个观测者（嵌套）。这是理解意识形态斗争、文明对话、跨物种认知的关键，  
然而又可以在抽象层面理解任何具有不同属性的观测者的理解将会是如何不同。

### 2.3 视界的构成

视界由三部分共同决定：

- 观测者自身性质
- 观测点（空间、时间、立场、角度）
- 观测镜（使用的工具）

### 2.4 视界的运动

- 随观测点移动而移动
- 随观测镜更换而重构
- 视界运动不是“视界消失”，而是“边界迁移”——**总有新的视界之外**

### 2.5 视界边界

视界边界不是一条锐利的线。  
它是一个**渐变带**——越靠近中心越清晰，越靠近边缘越模糊。

有些东西你确定在视界内（比如你手里的手机）。  
有些东西你确定在视界外（比如仙女座星系上的一只蚂蚁）。  
还有些东西，你只能说“好像在这里又好像在那里”——它们就在渐变带上。

这个“渐变”性质很重要。它解释了为什么你有时会觉得“前提在视界外”但又好像能触碰——它们可能在渐变带上。  
也会让你对一件事的可判性产生严重的质疑：万一这东西的可判性在视界之外呢？

### 2.6 视界之外

视界之外不可知、不可描述。  
你只能对其保持沉默。

但是我们知道他在视界内某种意义的存在，那就可以通过**添加约束**、**添加条件**，  
使其在视界内可判，从而将视界之外的一部分“补齐”。  
这个过程就是认知的扩展。  
请注意：补齐的不是“视界之外的真相”，而是“你添加约束后**生成的**可观测对象”。
在本理论不存在真相一说，只有符不符合视界之内的适应性。

## 第三章：构造描述集——你唯一能做的事

### 3.1 描述集的反向思路

我们的所有描述其实都是选择圈，最大从“元始”开始的，  
约束越少，抽象越高，描述范围越广泛。
我们添加约束，将抽象等级下降，使描述范围收敛，具体对象逐渐生成。

没有约束意味着绝对全集，里面是包含一切可能（包括矛盾）的。
添加约束就会划定子集，然后生成具体描述。

这个过程用到的所有约束，就构成一个描述集。

### 3.2 描述集的两种基本操作

- **解除约束**：扩大视界，将更底层的结构包含。  
  操作风险：也许你并不能脑补结果是什么，抽象过度还会解裂你的逻辑。
- **添加约束**：缩小视界，生成可见对象。  
  操作要求：约束必须来自某个来源（在具体应用中，来自周遭事实）。

### 3.3 约束的层级

- **底层约束**：在更大时空范围内稳定（如物理定律、逻辑法则）。解除它需要极大的认知跃迁。
- **上层约束**：易随时空环境变化（如文化习俗、意识形态）。解除它相对容易。

层级划分是相对的，取决于观测的时空尺度。

### 3.4 前提的真相

传统哲学追问“前提的前提”，试图找到“最终前提”。  
在约束操作下，这个问题的答案很简单：**前提不是你发现的实体，而是你添加的约束**。

这就是描述集的反向思维，不是世界是什么，是我怎么拟合世界。

追问“前提的前提”，就是在追问“约束的约束”——这可以无限后退，但每次后退都是一次解除约束的操作，最终抵达“元始”状态，然后你什么也说不出来。

就好比字典，你一个句子你不理解，你翻阅每一个词；
词你不理解，你翻阅字，但是为什么字之后就不能有再低的解释层级了呢？  
因为你要把解释给捅穿了，再往下就不是稳定的文字了，就是不能言表的纯粹的感性抽象，  
你再往下你还能说什么。

不要试图找到最终前提。  
那会让你大脑熵爆。  
适可而止，够用即可。

> 预测论事拓宽视界的一套抽象方法论（暂时的  
> 而且兼顾预备事变的任务。

_这个理论是极其依赖于已经探明的世界描述集的，也就是上一章，上头性质决定此论如何内容。_

### 4.1 预测的两种模式

- **推演**：在现有约束集内，基于连续性延伸当前趋势。  
  有效范围有限（类似求导外推），越远越模糊。
- **创造**：添加新约束，生成新的可能性框架，以作实验。
    > → 世界没给我的东西，我也许可以造出来。

### 4.2 预测三基本原则

**连续性原则**  
预测必须与现在连接，与历史连续。离开连续就是已经死亡。  
有效范围不会一下子扩充到全集（时空局限性）。

**穷举原则**  
不要放过时空视界内的任何线索、线索碎片、甚至是任何可能构成规律的东西。  
意识上必须最大化视界。  
但实际操作中，受限于使用者自身可承受的成本和具体问题的情况，他们会决定穷举到什么程度。

**留空原则（灵活原则）**  
事无常事。完满的、严密的预测总是抵挡不过永远的意外。  
视界和视界之外不是永远没有交互的。  
为减少意外，应穷举已知可能性，并允许变异（添加新约束）。但穷举永远无法完成，意外永远可能发生。

### 4.3 预测方法的认识论元原则

人类已有的预测方法（趋势外推、德尔菲、情景规划、TRIZ、层析因果、科幻科学等）虽然形式各异，但都隐含了一些没有明说的元原则：

1. **历史相似性假设**：未来会以某种方式重复过去的结构。  
   → 这是连续性的最低配版本。

2. **因果可解析性假设**：未来事件的原因可以被追溯到当前或过去的有限变量集合内。  
   → 这是穷举原则的可操作版本。

3. **多元可能性等价假设**：在信息不足时，多个不同的未来场景在逻辑上具有同等“候选资格”。  
   → 这是留空原则的积极版本（主动构造多元场景）。

4. **主体相对性假设**：预测结果依赖于预测者的立场、知识、价值。  
   → 这是使用者属性不可忽略原则的体现。

5. **有限可验证性原则**：一个预测至少在某些维度上可以在未来到来时被检验。  
   → 这是连续性原则的保障。

此外，还有一个更深层的**元原则零**：

1. **认知可迁移性假设**：从当前视界内获得的认知结构，当被迁移到视界边缘或未知区域时，在一定的认识连续范围内保持部分有效性。  
   → 这不是科学定律，而是预测行为得以发生的先验承诺。

### 4.4 检验

唯一检验是历史本身。人类无法干预，只能等待。  
至少一个预测样本命中，说明框架在某种程度“有用”——但不证明“正确”。  
我们只考虑他能拟合多少。

### 4.5 预测的终极态度

预测不是“看到未来”。  
预测是“在当前约束下，构造一个最可能的未来”。

不要追求永不犯错。  
追求犯错后能快速调整。

---

## 第七章：视界唯物主义——我们的立场

### 5.1 核心命题

1. **世界是物质的**，独立于意识存在。这是唯物主义的底线。
2. **任何对世界的认知，都受限于认知主体的视界**。视界由观测点、观测镜、观测者自身性质共同决定。
3. **视界不是固定的**——它随认知扩展而移动，但永远有视界之外。
4. **唯物主义不需要承诺“全知”**。承认视界之外的存在，不等于唯心；相反，这是对物质世界无限性的诚实态度。
5. **认知的任务不是在视界之外寻找绝对真理**，而是在视界之内，通过添加/解除约束，生成对当前实践有效的描述。

### 5.2 与经典唯物主义的区别

| 维度             | 经典唯物主义                 | 视界唯物主义                 |
| ---------------- | ---------------------------- | ---------------------------- |
| 可知论           | 世界原则上完全可知           | 世界局部可知，整体有视界之外 |
| 规律观           | 规律是客观的、可被完全认识的 | 规律是在视界内有效的约束     |
| 观测者           | 往往隐含“无干扰的客观观测”   | 观测者自身的性质不可忽略     |
| 预测             | 可以预测必然趋势             | 预测是连续+穷举+留空的操作   |
| 对“不可知”的态度 | 视为暂时的、待解决的         | 视为结构性的、相对永恒的     |

### 5.3 为什么它仍然是唯物主义？

因为：

- 视界之外是物质的，只是我们暂时（甚至永远）无法认知。
- 唯物主义不要求“全知”。
- “不可知”不是“不存在”。

### 5.4 一个口号

> **“视界之内，我们认真描述；视界之外，我们保持沉默——但不否认它的存在。”**

---

## 第八章：使用原则、红线与警示

### 6.1 原则

1. **锚定现实**：添加的约束必须来自周遭事实（视界内可验证的内容）。没有锚定，描述就会飘浮。
2. **连续操作**：每次解除或添加约束时，保持与现有认知的连续性。可以从现有认知一步步扩展，但不要跳跃到完全断裂的地方。
3. **穷举与留空**：在预测或构建时，尽可能穷举已知可能性（穷举原则），同时为未知可能性留出空间（留空原则）。穷举是为了减少意外，留空是为了接纳意外。
4. **工具理性**：选择用什么约束、生成什么描述，由发展的现实事实决定，而非理论本身。理论是工具，不是主人。

### 6.2 红线

1. **不要试图证明本理论**：没有证明一说，只有能不能拟合周遭。“证明”这个概念只有在结合其他理论（如科学方法论）时才出现。
2. **不要用它取代实践**：它生成描述，但选择和应用取决于现实判断。把理论当实践，等于把地图当土地。
3. **不要宣称“一切在视界内”**：任何时候都有视界之外的东西。宣称全知是自欺，也是思维崩溃的前兆。

### 6.3 警示单

| 违规操作             | 后果                  |
| -------------------- | --------------------- |
| 试图跳出视界描述视界 | 逻辑无限递归 → 熵爆   |
| 试图找到“最终前提”   | 无限后退 → 思想窒息   |
| 试图穷举全集         | 认知过载 → 抽象蒸发   |
| 混淆抽象层与具体层   | 自指混乱 → 逻辑虚无   |
| 忘记自己是观测者     | 架空自己 → 思想死亡   |
| 过度抽象化自己       | 违反连续性 → 自我瓦解 |

### 6.4 适用范围

- **适用于**：认知论探讨、世界观实验、跨文明想象、意识形态分析、逻辑底层重构。
- **不适用于**：需要精确预测的实务操作、需要确定性结论的科学实验（在这些领域应使用常规科学方法）。

这本书写到这里，应该结束了。

剩下的页面，是空白的。  
留给你自己去填充——用你的属性、你的视界、你的约束操作。

## 后记

这本书以后，也许  
你说“我再也不玩抽象了”，也许  
你说“哲学到头了”。

但你还会有意见，将以你具体的视界环境改写这个理论。  
你还会思考。  
你还会追问。

视界唯物主义不应是也不会是慵懒的主义，  
他们会是积极探索世界的，而且从不自欺欺人，  
除非视界大环境迫使他们放弃原先的唯物主义，但是他们记得  
**知道自己有限，但仍然行动。**  
他可能以后不会是唯物主义，但是永远应该是一个主动的主义。

这本书会发展。  
它的发展离不开别人——每一个使用它、质疑它、修改它的人，都会让它变得更丰富。

所以，这不是终点。  
他在作 永远的 绝对的相对无穷 的运动。
