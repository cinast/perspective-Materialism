## 本论性质

这不是世界观，也不是某某主义。  
它是一种元工具（元理论、元世界观），位于世界观之前。它不回答“世界是什么”，而是提供如何制造描述集，并用描述集构造世界的方法。

因而，本论包含多个抽象层级。

- I. **元构造层**：象层：通用理论，适用于任何使用者（包括假设的外星文明、AI等）。从“无约束”出发，描述所有可能的世界生成逻辑。**但本身不具备任何内容**。  
  **而且**必须给予一定的特性才能使用，否则不可能得到结果。

- IV. **具体实例层**：用元构造生成的世界

- III. 不过本论自身是指导性质的，**所以还有一层元构造的方法论层级**。

但是**元构造层本身不具备任何内容**，因为不存在具体的东西，不存在限制，他**需要一些外来的预设**，比如**一定存在的**本论使用者的自带性质，或者更具体的但可能存在的，使用者的目的，才能运转。  
而且试图脱离使用自身性质构造事物，往往会造成架空自己然后思想窒息的局面。

既然本论的结果如何取决于使用者，那么**使用者自身的因素是不可以忽略的**，**无论使用者是谁**。  
就此论编写现处的时空坐标，这个理论的文本面向人类。

那么实际上

- II. 还存在一层隐性的抽象层级，在元构造层与具体示例层中间，那就是 **使用者具体的元构造层**

此外**构造和实例两层必须分开讨论**，否则会产生严重的抽象层级混乱和逻辑虚无；
但 使用者具体的元构造层 和 元构造层 一般来说 使用者已经默认具体化 元构造层了，而且大多数时候用的也是具体化的元构造层。

## 第一部分：元构造基本（通用）

_构造世界，或者认识世界_

### 1. 世界原型

    你需要一个世界模胚制（原型）造你的世界，或拟合你所处的世界。

    这个原型尚不具备任何具体内容，只有一些基础要素。
    你可以随意构造任何基础要素的世界，不存在任何限制，完全没有具体指导。此处原本不具备任何描述。

    但无一例外：至少原命题的存在是在本论使用者的视界之内的（见后文“视界”）。

     - 描述集

         你需要一系列基础要素制定你的世界原型，这些要素就是一些命题，制定世界行为的边界，约束它，使其发生机械运动。但需要填入命题。

         具体的描述集就是世界观。  \
         抽象的、填空的描述集就是世界原型。

         按照通常，命题由你的属性给予。

      - 约束
         所有描述都从无约束（绝对全集，最大范围）开始。添加约束，抽象等级下降，描述范围收敛，具体对象生成。

         无约束 = 绝对全集 = 包含一切可能（包括矛盾）  \
         添加约束 = 划定子集 = 生成具体描述

      - 基础要素

         决定你的世界原型会有什么特性，什么内容出现，决定他的运转，以及他的意义。 \
         内容任意。

通过上述操作制造的一系列约束，形成抽象描述集。抽象描述集可以用于构造世界、构造主义。  
但描述集本身不预设立场。

### 2. 视界（通用定义）

#### 2.1. 观测者模型（描述者模型）

模型三要素：世界、望远镜、观测者。

- 世界：被观测的对象整体（可能是整个宇宙、某个子系统、或任何研究对象） \
  使用时注意规则如下：
    - **必须绝对全集**：涵盖一切内容，没有特殊遗漏。 \
      若果存在意外事物，那应改用“视界”概念限定之。

- 望远镜：观测工具。包括物质工具（显微镜、探测器）和思想工具（语言、理论、意识形态、数学 框架）。

- 观测者：进行观测的主体，在世界内。  
   若果在世界外，就与世界没有关系，就不用观测一说。 \
  观测者自身的性质（位置、运动、认知能力）会影响视界。

关键自指：本理论的使用者本身就是观测者。使用者不能忽视自己的存在——因为使用者是这个模型的 一部分。使用者必须承认，自己是世界的产物，此论的性质不能离开使用者自身的性质。

#### 2.1 视界与世界

- 视界：对于某个观测者而言，可观测、可描述、可接触的范围的集合。视界具有运动性，会变化。
- 视界是世界的子集（世界 ≥ 视界）。
- 视界之内：可观测、可描述。
- 视界之外：对于该观测者是“尚未添加约束”的状态——不可知、不接触、不可描述。 \
   你只能对其保持沉默。

#### 2.2 视界的构成

- 观测者自身性质：观测者的物理属性、认知能力、历史位置等。
- 观测点：观测时的位置（空间、时间、立场、角度）。
- 观测镜：使用的工具（物质或思想）。
  三者共同决定视界边界。

#### 2.3 视界的运动

- 随观测点移动而移动。
- 随观测镜更换而重构。
- 视界运动不是“视界消失”，而是“边界迁移”——总有新的视界之外。

### 3. 抽象观测论：接入视界之外

视界之外原本不可知。但可以通过添加约束，将视界之外的一部分“接入”视界内部 \
这个过程就是认知的扩展。

#### 3.1 两种基本操作

- 解除约束：扩大视界，看到更底层的结构。操作风险：可能引发逻辑递归或熵爆。
- 添加约束：缩小视界，生成可见对象。操作要求：约束必须来自某个来源（在具体应用中，来自周 遭事实）。

#### 3.2 约束的层级

- 底层约束：在更大时空范围内稳定（如物理定律、逻辑法则）。
- 上层约束：易随时空环境变化（如文化习俗、意识形态）。
- 层级划分是相对的，取决于观测的时空尺度。

千年科学史，百年唯物主义，历史拷打，

## 1\*. 现在的视界内的世界的世界描述集：唯物主义世界观

1. 时空连续，前后关联，运动不断
   我们有历史一说，视界范围内历史连续，前后呼应关联不中断。可以大体上用“可导”来描述局部。
2. 无限的绝对和相对的有限
   人类无法证明宇宙有限或可穷举，但缩小命题范围可以发现“绝对的相对之有限”（或“相对的绝对”）。

   无论你如何臆想，只要你不动，世界照样按部就班。即使暂时中断你的意识，你观察的世界总是 与先前记忆的世界存在关联，且可以通过历史“考古”补全。视界内唯物已全胜。 \
   但是还不建议现在将此条目抽象到上面，它会蒸发掉你的因果律。（见[唯心主义不适用此工具](#唯心主义不适用此工具)）

## 2\*. 视界论（针对人类）

与通用定义一致，但增加以下具体内容：

- 视界之外：人类不知道，不能描述，不能从中获得任何命题或证明，没有答案。
- 通过连续性在局部证明某个命题，就能在连续的领域获得小误差的答案，并在将来涉足时验证。
- 世界上不存在绝对的“对错之论”。对于人类，只有符不符合周遭（实践检验）。

## 预测论（从约束操作衍生）

### 预测的两种模式

- 推演：在现有约束集内，基于连续性延伸当前趋势。有效范围有限（类似求导外推），越远越模糊。
- 创造：添加新约束，生成新的可能性框架。 \
  这不是“预测未来”，而是“构建可能世界”。

### 预测原则

#### 穷举原则

不要放过时空视界内如何的线索、线索碎片、甚至是任何可能构成规律的东西。

#### 留空原则（灵活原则）

事无常事，完满的、严密的预测总是抵挡不过永远的意外。 \
视界和视界之外不是永远没有交互的。

#### 连续性原则

- 预测必须与现在连接，与历史连续。离开连续就是已经死亡。
- 有效范围不会一下子扩充到全集（时空局限性）。
- 为减少意外，应穷举已知可能性，并允许变异（添加新约束）。但穷举永远无法完成，意外永远可 能发生。

### 检验

唯一检验是历史本身。人类无法干预，只能等待。至少一个预测样本命中，说明框架在某种程度“有 用”——但不证明“正确”。

## 使用原则与警示（针对人类使用者）

### 原则

1. 锚定现实：添加的约束必须来自周遭事实（视界内可验证的内容）。
2. 连续操作：保持与现有认知的连续性，避免断裂。
3. 穷举与变异：在预测或构建时，使用穷举的视角，并留出变异空间以接纳强烈的变化。
4. 工具理性：选择用什么约束、生成什么描述，由发展的现实事实决定，而非理论本身。

### 警示

1. 不要试图证明本理论：他没有证实的说法。
2. 不要用它取代实践：它生成描述，但选择和应用取决于现实判断。
3. 不要忘记视界：任何时候都有视界之外的东西。宣称“一切都在视界内”是自欺。 \
   试图跳出视界描述视界、试图找到“最终前提”、试图穷举全集——都会导致思维混乱。
4. 警惕自指混乱：使用本理论时，必须明确当前是在“抽象元工具”层还是“具体应用”层。两层混淆 会使抽象混乱。

适用范围

- 适用于：认知论探讨、世界观实验、跨文明想象、意识形态分析、逻辑底层重构。
- 不适用于：需要精确预测的实务操作、需要确定性结论的科学实验（在这些领域应使用常规科学方 法）。

# 唯心主义不适用此工具

在抽象元工具层，任何世界观（包括唯心）都可以使用——因为工具本身不预设立场。 \
但在具体应用层 （基于唯物主义视界），如果你坚持唯心主义的核心命题（如意识决定物质），那么工具的许多操作（如“锚定现实”“约束来自周遭”）将无法一致地执行。 \
除非真有个唯心主义世界在那，但是那样因果律就几乎覆灭了，它本身将会立足于一个极度不稳定的世界，并且其本身的适应能力完全不与其是一个级别。 \
因此，对于人类当前视界内的具体应用，我们默认采用唯物主义立场。 \
唯心主义者若想使用，必须先在抽象层构建一套自洽的唯心主义版本——但那不是本理论的重点， \
而且伦理上决不允许，我们知道这会被如何使用的。
